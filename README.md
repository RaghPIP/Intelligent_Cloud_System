# Phase 1: Data Preprocessing

## Quick Start

```bash
# Install dependencies
pip install Flask pandas

# Run Flask app
python3 flask_app.py

# Open browser
http://localhost:5000
```

## Files

- `flask_app.py` - Main Flask application with UI
- `phase1_preprocessing.py` - Data preprocessing script
- `data/` - Upload datasets here (CSV files)
- `Phase1_Data_Preprocessing.md` - Complete documentation

## Usage

1. Click "Data Preprocessing" in sidebar
2. Select number of datasets (1-5)
3. Upload CSV files
4. Click "Start Data Preprocessing"
5. View results on dashboard
